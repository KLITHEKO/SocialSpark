Hi, this is the social spark readme, a mobile application designed to create meaningful social connections throughout the day. The application is entirely made on android studio with Kotlin using if statements for the logic and xml for the user interface. 

Tailored Suggestions:
Prompts are generated based on the time of day presenting opportunities for meaningful interaction across the entirety of the day.

Visual look/design
The interface uses an artisan sage and slay colour palette, using earthy tones with flat material design principles. Using subtle borders and soft shadows to create a clean and modern aesthetic.
Fonts:
Lora (Serif)
Robot (sans-serif)
